MBSYS is a premium, high-performance platform dedicated to enterprise IT infrastructure, security solutions, and modern workspace optimization. Built with a focus on visual excellence and seamless brand connectivity.

---

## 💎 Visual Experience

- **Dark-Locked Premium UI**: A curated Slate-950 aesthetic designed for professional authority.
- **Meteor Shower Background**: Dynamic, GPU-accelerated canvas animations that bring the digital workspace to life.
- **Seamless Brand Ripple**: Intelligent cross-brand transitions to the Hunar design studio with synchronized color-state handoffs.
- **Micro-Interactions**: High-frequency feedback loops using Framer Motion for a tactile digital feel.

## 🚀 Technical Architecture

- **Engine**: [Vite](https://vitejs.dev/) + [React 19](https://react.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/) (Cutting-edge engine for rapid, performant design)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **SEO**: Fully optimized metadata, semantic HTML5, and automated sitemap generation.

## 🛠️ Local Development

### Prerequisites
- Node.js (v18+)
- npm / yarn

### Installation
1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables (if required for Gemini AI features):
   Create a `.env` file and add:
   ```env
   GEMINI_API_KEY=your_key_here
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

The site is optimized for **Netlify** with a custom `netlify.toml` configuration for clean routing and performance headers.
